# investimento2026

Sobre o projeto
O Investimentos é um aplicativo desenvolvido para simular o crescimento de um investimento ao longo do tempo.

A proposta é mostrar uma alternativa ao financiamento: ao invés de comprar parcelado, o usuário pode investir um valor mensal e acompanhar quanto terá acumulado no futuro.

O usuário informa:

Valor mensal a ser investido
Número de meses
Taxa de juros mensal
E o sistema retorna:

Montante acumulado sem juros
Montante acumulado com juros compostos
Prints das telas
  

Funcionalidades
Cálculo do valor acumulado sem juros
Cálculo com juros compostos
Simulação por período (meses)
Interface simples e intuitiva
🚀 Tecnologias utilizadas
Flutter
Dart
