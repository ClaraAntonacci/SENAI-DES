# Calculadora de Bitola - bitola2026

Aplicativo desenvolvido em Flutter para calcular a bitola de cabos elétricos de forma simples e prática.

## Sobre o projeto

A Calculadora de Bitola permite que o usuário informe a distância e a corrente elétrica para obter a bitola recomendada de cabos em instalações residenciais.

A proposta é auxiliar no dimensionamento correto dos cabos, garantindo mais segurança e eficiência.

O usuário informa:

Distância em metros  
Corrente elétrica (em amperes)  

E o sistema retorna:

Bitola recomendada para 12V  
Bitola recomendada para 220V  

## Funcionalidades

- Inserir distância em metros  
- Informar corrente elétrica  
- Cálculo automático da bitola  
- Resultado para diferentes tensões (12V e 220V)  
- Interface simples e intuitiva
