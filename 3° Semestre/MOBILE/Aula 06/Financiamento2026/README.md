# simulador_financiamento2026

## Sobre o projeto
O Simulador de Financiamento é um aplicativo desenvolvido para calcular valores de financiamentos de forma simples e prática.

A proposta é permitir que o usuário insira os dados do financiamento e visualize rapidamente o valor das parcelas e o total a ser pago.

O usuário informa:

Valor do financiamento  
Taxa de juros mensal (ex: 0.02)  
Número de parcelas  
Taxas e custos adicionais  

E o sistema retorna:

Valor da parcela  
Valor total a ser pago
